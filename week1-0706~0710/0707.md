# Chapter 4 함수

파이선에서 함수는 일급 객체

- 함수도 하나의 자료형
- runtime에 생성하는 것이 가능
- 변수에 대입이 가능
- 함수를 다른 함수의 매개변수로 사용하는 것이 가능
- 함수를 함수의 결과로 리턴하는 것도 가능

## built in 함수

dir(__builtins__)를 호출하면 built in 함수를 출력할 수 있다.

함수의 도큐먼트 확인

이름(매개변수 나열) → 리턴되는 데이터

- 리턴되는 데이터가 없으면 None
- 매개변수 앞에 *이 있으면 데이터를 여러개 대입할 수 있다

3) 사용자 정의 함수

⇒ 함수 생성

```python
def 함수이름(매개변수 나열):

	함수가 수행할 내용

	[return 데이터]
```

- return 하는 문장이 없으면 None을 리턴하는 것으로 간주
- 매개변수는 함수를 호출할 때 필요한 데이터

⇒ return

- 함수의 수행을 종료하고 호출하는 곳으로 이동하는 제어문
- return 할 때 하나의 데이터를 가지고 리턴할 수 있다.
- 최근에는 함수의 리턴 타입을 적기도 한다.
    
    ```python
    def 함수이름(매개변수 나열) -> 리턴되는 데이터의 타입:
    
    	함수가 수행할 내용
    
    	[return 데이터]
    ```
    

---

파이선 함수는 return문에 여러개의 데이터 타입을 보낼 수 있는데 사실 **튜플**의 형태로 보내는 것이다.

## 순수함수

- 함수의 실행이 외부 상태에 영향을 미치지 않는 경우

함수 내부에서 매개변수를 직접 수정하지 않고 리턴을 해서 결과를 사용하도록 해주는 것을 권장

- 함수의 입력이 같으면 항상 동일한 결과를 만들어 주는 경우

랜덤이나 현재 시간 같은 데이터를 이용해서 결과를 만들지 않도록 하는 것을 권장

- 순수 함수가 아닌 함수를 비순수 함수라고 한다.

### 데이터 처리와 관련된 함수
- **map**: 데이터를 변환해주는 함수  
데이터 변환을 함수와 데이터 목록을 넘겨주면 데이터 목록의 데이터를 하나씩 함수에 대입해서 리턴받은 결과로 데이터의 모임을 만들어주는 함수
대입해야 하는 함수는 하나의 매개변수를 받아서 리턴하는 함수여야 한다.
<br>

- **filter**: 데이터를 골라내는 함수
데이터 추출을 위한 함수와 데이터 목록을 넘겨주면 데이터 목록의 데이터를 하나씩 함수에 대입해서 True를 리턴받은 데이터만 가지고 데이터 모임을 만들어준다. 
<br>

- **reduce**: 데이터를 가지고 연산하는 함수
기본 내장함수가 아니다. 
functools 라는 패키지의 함수로 존재
2개의 매개변수를 가지고 데이터를 리턴하는 함수를 대입하면 되는데 첫번째 매개변수는 앞에서 수행한 함수의 결과
